Markdown is a lightweight markup language that you can use to format text. It’s commonly used on platforms like GitHub for documentation. Here’s a quick guide to some basic Markdown syntax for creating headings, bold text, and other common formatting.

### Headings

You can create headings by using hash symbols (`#`). The number of hash symbols indicates the level of the heading:

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```

### Bold and Italics

- **Bold**: Use double asterisks (`**`) or double underscores (`__`) around the text.

  ```markdown
  **This is bold text**
  __This is also bold text__
  ```

- *Italic*: Use single asterisks (`*`) or single underscores (`_`) around the text.

  ```markdown
  *This is italic text*
  _This is also italic text_
  ```

### Lists

- **Unordered Lists**: Use asterisks (`*`), plus signs (`+`), or hyphens (`-`).

  ```markdown
  - Item 1
  - Item 2
    - Subitem 1
    - Subitem 2
  ```

- **Ordered Lists**: Use numbers followed by periods.

  ```markdown
  1. First item
  2. Second item
     1. Subitem 1
     2. Subitem 2
  ```

### Links

To add hyperlinks:

```markdown
[Link text](http://example.com)
```

### Images

To add images:

```markdown
![Alt text](http://example.com/image.jpg)
```

### Code

- **Inline Code**: Use backticks (`` ` ``).

  ```markdown
  Here is `inline code`.
  ```

- **Code Blocks**: Use triple backticks (```` ``` ````) or indent with four spaces.

  ```markdown
  ```
  This is a code block
  ```
  ```

### Blockquotes

Use the greater-than symbol (`>`).

```markdown
> This is a blockquote.
```

### Horizontal Lines

Use three or more dashes (`---`), asterisks (`***`), or underscores (`___`).

```markdown
---
```

### Tables

To create tables, use pipes (`|`) and hyphens (`-`).

```markdown
| Header 1 | Header 2 |
|----------|----------|
| Row 1 Col 1 | Row 1 Col 2 |
| Row 2 Col 1 | Row 2 Col 2 |
```
